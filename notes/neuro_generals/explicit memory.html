Explicit memory

Aka

declarative memory

See also

memory#types

implicit memory

Definition

memory for knowledge/facts

can be recalled by conscious effort

reported verbally

flows automatically when doing things

Tulving - divided explicit memory further into:

episodic

semantic

see patient HM#impaired

Mechanism

explicit memory first acquired by processing in these polymodal association cortices:

prefrontal cortex

limbic system

parieto-occipital-temporal

then conveyed in series to

parahippocampal cortex

perirhinal cortex

then

hippocampus#mechanisms

entorhinal cortex

dentate gyrus

hippocampus

subiculum

back to

entorhinal cortex

then sent back to

parahippocampal cortex

perirhinal cortex

finally back to polymodal association cortex

prefrontal cortex

limbic system

parieto-occipital-temporal

Clinical

damage to hippocampus or polymodal association areas produces clear impairment of explicit memory

see hippocampus#clinical

damage to perirhinal cortex, parahippocampal cortex or entorhinal cortex without damage to hippocampus:

larger defiict in explicit memory (e.g. object recognition) than specifically lesioning the hippocampus

see patient HM, patient NA and patient RB

damage to amygdala alone does not impair explicit memory
