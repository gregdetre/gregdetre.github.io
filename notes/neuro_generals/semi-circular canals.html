Semi-circular canals

Part of

vestibular system

Function

angular acceleration

Details

fluid filled hoops - tell you which way the fluid is flowing in the canals

3 canals - 3 dimensions - each detects motion in a single plane

continuous endolymph-filled hoop

hair cells sit in a swelling at the base (ampula)

angular acceleration causes the endolymph to slosh around, deflecting the hair cells

the hair cells are polarized

push one way -> excitation

push the other way -> inhibition

if both push at once

  • > vertigo/nausea

brain eventually adapts if one vestibular nerve is cut

wired up to oculomotor system, so that head movements -> eye movements

see vestibulo-ocular reflex
