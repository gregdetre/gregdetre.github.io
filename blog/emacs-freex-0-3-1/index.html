Emacs freex 0.3.1

Woohoo! The first public release of Emacs freex minor mode is now available (v 0.3.1) (co-authored with Per Sederberg).

The help is currently scanty, but there's a starter tutorial and some installation instructions. If you do get it working, I'd love to hear from you.
