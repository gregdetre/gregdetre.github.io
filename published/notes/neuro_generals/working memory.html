Working memory

Definition

special short-term memory store

Conists of

3 components

central executive (attentional control system)

prefrontal cortex

phonological loop (or articulatory loop)

for words and numbers

may involve Brocas area

visuospatial sketchpad

visual properties and spatial location

the phonological loop and visuospatial sketchpad are rehearsal systems, i.e. activation-based memory systems

rapidly decaying memory trace
