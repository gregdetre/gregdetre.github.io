Definition

learning = acquiring knowledge about the world

memory = encoding, storing and retrieving that knowledge

also defined as any persistent effect of experience

engram - physical change in the brain corresponding to learning

Types of memory

declarative (explicit memory)

procedural (implicit memory)

[circular definition - the stuff that HMM can't do is declarative]

Processes

encoding

attending to the information, and associating it meaningfully and systematically with knowledge already well-established to integrate the new with the old. stronger when well-motivated

consolidation

alter the newly stored and still labile infomration to make more stable for long-term

epxression of genes and synthesis of new proteins, causing structural changes that store memory stbaly over time

storage

retaining memories over time

retrieval

bringing information back together from multiple storage sites
