Figure that doesn't steal focus

From Sean Polyn:

I got fed up with the tendency for calls to the "figure" command in matlab to steal focus from all other applications (such as when you are running a big loop making lots of figures.

So I googled: "matlab foreground figure" and the first hit gave this code (really just a single "set" command embedded in a little script) for SILENT FIGURES. So I'm happy.
