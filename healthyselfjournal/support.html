Getting Started

Quick Setup

  1. Download the app from TestFlight (iOS).

  2. Grant microphone permission when prompted.

  3. Add your API keys in Settings (required for AI features) - see below.

  4. Start journaling by pressing the record button.

Getting API Keys

Healthyself requires your own API keys for transcription and AI features:

OpenAI API Key (for transcription)

  1. Visit platform.openai.com

  2. Sign up or log in

  3. Go to API Keys section

  4. Create a new key

  5. Copy and paste into Healthyself Settings

Anthropic API Key (for AI dialogue)

  1. Visit console.anthropic.com

  2. Sign up or log in

  3. Go to API Keys section

  4. Create a new key

  5. Copy and paste into Healthyself Settings

Important: Keep your API keys secure. Healthyself stores them locally on your device and never transmits them to our servers.

Features

Voice Recording

  • Press the microphone button to start recording

  • Speak naturally - there's no time limit

  • Press stop when you're done

  • Your audio is saved locally on your device

AI-Powered Reflection

  • After recording, your speech is transcribed to text

  • AI generates thoughtful questions based on what you shared

  • Continue the dialogue to deepen your reflection

Privacy & Security

  • All journal entries stored locally on your device

  • Your API keys never leave your device

  • Optional Dropbox sync to YOUR account only

  • No data collection or tracking

Dropbox Sync (Optional)

  1. Go to Settings

  2. Tap "Connect Dropbox"

  3. Authorize the app

  4. Your sessions will sync to a folder in your Dropbox

Troubleshooting

"Transcription failed"

  • Check your OpenAI API key is entered correctly

  • Verify you have API credits in your OpenAI account

  • Check your internet connection

"AI responses not working"

  • Check your Anthropic API key is entered correctly

  • Verify you have API credits in your Anthropic account

  • Check your internet connection

"Microphone not working"

  • Go to iOS Settings > Privacy > Microphone

  • Ensure Healthyself has permission

  • Restart the app

"Lost my API keys"

  • API keys are stored securely on your device

  • If you delete the app, you'll need to re-enter them

  • Check your OpenAI/Anthropic accounts to view or regenerate keys

Frequently Asked Questions

Why do I need my own API keys?

We believe in complete privacy. By using your own API keys, your journal content goes directly from your device to the AI service - we never see it.

How much will API usage cost?

  • OpenAI: Typically $0.006 per minute of audio

  • Anthropic: Approximately $0.01-0.03 per journal session

  • Most users spend $1-5 per month with regular use

Is there an Android version?

Android support is planned for future release. Currently iOS only.

Can I use the app offline?

Voice recording always works offline.

Cloud transcription and LLM features require internet connection.

Local on-device AI is in development.

How do I export my journal entries?

In the current version, your transcripts are saved locally and synced to Dropbox (if connected).

Export features are coming soon.

Is my data backed up?

  • Local backups via iOS system backups

  • Optional Dropbox sync for cloud backup

  • We recommend enabling Dropbox sync for safety

Privacy Policy

Healthyself is committed to your privacy:

  • No data collection: We don't collect or store your journal entries

  • Your API keys: Used directly from your device, never sent to us

  • No tracking: No analytics, no ads, no user tracking

  • Open source: Our code is transparent and auditable
